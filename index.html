Welcome to Chronos

A quick walkthrough of the key features.

📋
Create tasks on the right
Hit N or the + Add task button. Set a name, duration, category, and optionally a recurring schedule (weekly or every 2 weeks on specific days).
🖱
Drag tasks onto the calendar
Drag any task card from the right panel onto an empty time slot on the left. The drop preview shows the exact time. A red outline means a conflict — find a free slot.
✏️
Draw events directly on the calendar
Click and drag downward on any empty calendar space to draw a new event. A quick-create dialog will pop up when you release.
⚙️
Edit & manage events
Double-click an event to edit its name, time, category, and notes. Right-click for a context menu with duplicate, lock, mark done, and remove.
↩
Move & remove events
Drag events to reposition them. Resize by dragging the bottom edge. To remove, drag an event to the right panel — the top zone returns it to the task list; the bottom zone deletes it.
🔄
Recurring events & biweekly view
Tasks can repeat every week or every 2 weeks on chosen days. Use W1 / W2 in the top-right to toggle which biweekly group is visible.
💾
Save & export
Save produces a code you can paste back into Load on any device. Apple Cal exports a date-range .ics file for import into any calendar app. PDF opens a print-ready view.
📊
Stats tab
See hours by category and by day. Adjust your daily hour target (changes the bar under each date). Manage custom categories. Set calendar display hours.
Keyboard shortcuts:  N new task  ·  L bulk import  ·  T today  ·  ←→ weeks  ·  Ctrl+Z undo  ·  ? shortcuts

✦ AI Assistant

🤖 Works best with Claude
Chronos was built with Claude AI and the prompt format is optimised for it. Claude understands the bulk format perfectly and can analyse your pasted schedule. Any LLM works, but Claude gives the best results.
What AI can do for you:
📝
Generate task lists — describe your week or goals, ask the AI to format them as Chronos bulk import text. Copy and paste straight into the List (⋮) button.
📊
Analyse your schedule — paste your saved schedule code (from the Share Schedule tab) into a chat and ask AI to review it for balance, overload, or missing tasks.
🔄
Optimise & reschedule — paste your schedule, explain constraints ("I need 2h focus time every morning"), and ask the AI to suggest a revised task list you can re-import.
💡
Estimate durations — list your projects and ask AI to estimate realistic time blocks, then format them for import.
Workflow tip: Open Claude at claude.ai side-by-side with Chronos. Paste the AI Prompt once to set context, then have a back-and-forth conversation to refine your schedule — import new lists as you iterate.
